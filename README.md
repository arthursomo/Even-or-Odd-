# Even or Odd Calculator
 
A simple Python script that multiplies a list of numbers provided by the user and checks whether the result is even or odd.
 
## What it does
 
The user enters a sequence of numbers separated by spaces. The program:
1. Multiplies all the numbers together
2. Checks whether the final result is even or odd
3. Prints the outcome to the terminal
It also handles invalid input (e.g. if the user types a letter instead of a number) without crashing.
 
## How to run
 
```bash
python even_odd_calculator.py
```
 
## Example usage
 
```
Enter numbers separated by spaces: 1 2 3 4 5 6 7
5040 is even
```
 
Invalid input example:
 
```
Enter numbers separated by spaces: 1 2 c 4
Error: please enter only numbers separated by spaces.
```
